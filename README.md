# Peer Slot Exchange Project

## Project Info

This is a React + TypeScript project using Vite, Tailwind CSS, and shadcn-ui.

## Getting Started

You can edit and run this project locally by following these steps:

1. **Clone the repository**

git clone <YOUR_GIT_URL>

2. **Navigate to the project directory**

cd <YOUR_PROJECT_NAME>

3. **Install dependencies**

npm install

4. **Start the development server**

npm run dev

The development server supports hot-reloading, so your changes will reflect instantly in the browser.

## Project Structure

- **src/**: Contains all your React components, pages, and styles.
- **public/**: Static assets like images and icons.
- **vite.config.ts**: Vite configuration.
- **tailwind.config.js**: Tailwind CSS configuration.

## Technologies Used

- **React**
- **TypeScript**
- **Vite**
- **Tailwind CSS**
- **shadcn-ui**

## Deployment

You can deploy this project to any hosting provider that supports static React apps, such as:

- Vercel
- Netlify
- GitHub Pages
- Firebase Hosting

Build the project for production:

npm run build

This will create a `dist` folder containing the production-ready files.

## Custom Domain

You can configure a custom domain with your hosting provider. Point the domain to the `dist` folder or the hosting platform as per their instructions.
